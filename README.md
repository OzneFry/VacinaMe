# VacinaMe

## Desenvolvimento orientado por specs

Este projeto agora conta com uma estrutura leve de Spec-Driven Development em [specs](specs).

- Use [specs/template.md](specs/template.md) como modelo para novas features.
- Veja [specs/examples/api-playground-spec.md](specs/examples/api-playground-spec.md) como exemplo prático.
- Antes de implementar uma nova funcionalidade, crie uma spec simples e mantenha o processo enxuto.
